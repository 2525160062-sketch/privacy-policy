# Política de Privacidad

Fecha de última actualización: 2025

Esta Política de Privacidad describe cómo se recopila, utiliza y protege la información personal de los usuarios dentro de este servicio/aplicación/sitio web (en adelante, “el Servicio”).

## 1. Información que recopilamos

El Servicio puede recopilar los siguientes tipos de información:

- **Información proporcionada por el usuario:** como nombre, correo electrónico, o cualquier otro dato ingresado voluntariamente.
- **Datos técnicos:** como dirección IP, tipo de dispositivo, sistema operativo, idioma, y datos de uso o interacción dentro del Servicio.
- **Cookies y tecnologías similares:** utilizadas para mejorar la experiencia del usuario.

## 2. Uso de la información

La información recopilada puede utilizarse para:

- Proveer y mantener el Servicio.
- Mejorar la experiencia del usuario.
- Analizar el uso del Servicio.
- Comunicarnos con el usuario (notificaciones, soporte, actualizaciones, etc.).

## 3. Compartición de información

Tu información **no se comparte** con terceros, excepto en los siguientes casos:

- Cuando sea necesario para operar el Servicio (proveedores de hosting, análisis, etc.).
- Cuando la ley lo requiera.
- Para proteger la seguridad, derechos o integridad del Servicio.

## 4. Seguridad de los datos

Implementamos medidas de seguridad razonables para proteger la información personal.  
Sin embargo, ningún método de transmisión o almacenamiento es 100% seguro.

## 5. Derechos del usuario

El usuario puede solicitar:

- Acceso a sus datos personales.
- Corrección o eliminación.
- Limitación del uso de sus datos.

Para ejercer estos derechos, el usuario puede contactarnos mediante el medio proporcionado por el Servicio.

## 6. Cambios a esta política

Esta Política puede actualizarse periódicamente. Se notificará al usuario si se realizan cambios importantes.

## 7. Contacto

Para dudas o solicitudes relacionadas con esta Política de Privacidad, puedes ponerte en contacto con nosotros a través del método disponible en el Servicio.
